# mp-launch-checklist.md — 里程碑 4 前置详细清单（上线合规）

> 全是"花钱 + 等审批"的现实流程，办一项勾一项。可在里程碑 2~3 开发期间并行启动以省等待。
> 政策/价格会变，最终以官方页面为准（执照=当地市场监管局；备案=工信部 beian.miit.gov.cn + 云厂商备案系统）。
>
> **依赖顺序**：①营业执照 →（②③④）；③备案 需 ①执照 + ②服务器都就绪。

---

## ① 办营业执照（个体工商户）

**为什么选个体工商户而非公司**：注册简单、无注册资本要求、记账报税最简（可申请定期定额），完全够小程序主体用。

- [ ] 准备：身份证；字号名称（如"XX工作室/XX商行/XX科技中心"）；经营范围（勾"技术服务、技术开发、软件开发、信息咨询"等）；经营场所（多数地区住宅地址可作个体户场所，部分地区"住所申报承诺制"免房产证明，具体看当地）。
- [ ] 办理（二选一）：
  - 线上：当地「一网通办 / 政务服务网」或「电子营业执照」小程序、微信/支付宝的工商登记入口；很多地区个体户可全程网办、免费。
  - 线下：当地市场监督管理局 / 政务服务中心窗口。
- [ ] 领执照（电子版即可用于后续注册/备案）。
- [ ] （按需）刻章：部分地区免费赠章，否则几十元。

**成本**：执照免费，刻章可能几十元。**时间**：线上快的 1~3 个工作日。
**后续义务**：有记账报税义务，个体户可申请"定期定额"，很简单；可自行或找代账（几十~一两百/月）。

---

## ② 买国内云服务器（备案要用）

- [ ] 选厂商：**腾讯云**或**阿里云**（备案体系最成熟，和小程序生态近）。
- [ ] 选机型：**轻量应用服务器**最便宜够用（1~2 核 2G）；新用户常有优惠，约几十~一百多元/年。
- [ ] 选地域：**国内**（上海/广州/北京等）。⚠️ **不要选香港/海外**——海外节点不能用于工信部备案。
- [ ] 购买时长：**买满 1 年**。⚠️ 备案通常要求服务器剩余时长 ≥3 个月，买太短备案不让用。

**成本**：约几十~一百多元/年。**备案本身免费。**

---

## ③ 域名备案（ICP 备案）

**第 0 步——先查当前状态**（决定走哪条路）：
- [ ] 到 `beian.miit.gov.cn` →「公共查询」→ 查 `soultranslator.cn`。
  - 查不到 → 走**首次备案**。
  - 查到但是别人的信息（老备案）→ 走**备案转移 / 变更接入**到你的主体。
  - 查到且是你本人 → 极少见，基本可直接用。
- [ ] 确认 .cn 域名**实名认证**已完成（过户一般已实名，去聚名网控制台确认）。

**备案流程**（在②的云厂商「备案控制台」里全程操作）：
- [ ] 在云厂商备案系统发起备案，关联②买的服务器。
- [ ] 填主体信息：用①的营业执照（=企业/个体主体）、负责人身份证、域名信息。
- [ ] 上传材料：营业执照、身份证、域名证书，按提示填核验单/承诺书。
- [ ] App 人脸核验（云厂商 App 扫脸）。
- [ ] 云厂商初审（约 1~2 天）→ 提交所在省**通信管理局**。
- [ ] 管局审核：约 **7~20 个工作日**（可能有电话回访，留意接听）。
- [ ] 通过 → 下发备案号。

**成本**：免费。**时间**：**最长的一环，1~3 周**，所以越早并行启动越好。
**注意**：小程序场景主要是给**后端域名**（如 `api.soultranslator.cn`）备案/能用即可。

---

## ④ 注册小程序正式账号（替换测试号）

> 与③可并行。注册账号**不需要域名/备案**，只需①的执照。

- [ ] 打开 `mp.weixin.qq.com` → 注册 → 选「小程序」。
- [ ] 填邮箱、设密码（一个**没注册过公众号/小程序**的邮箱）。
- [ ] 主体类型选 **企业 / 个体工商户**，填①的营业执照、走对公或法人验证。
- [ ] 主体审核通过 → 账号开通 → 拿到**正式 AppID**。
- [ ] 在代码 config 里把测试号 AppID 换成正式 AppID。
- [ ] 在小程序后台「开发设置 → 服务器域名」填入备案好的后端域名（如 `https://api.soultranslator.cn`）。
- [ ] 选好**服务类目**（情感/AI 内容相关，按后台引导选，可能要补资质说明）。

**成本**：个体/企业主体注册小程序**免费**（注：若涉及微信支付等另说，MVP 不用）。

---

## 建议并行时间线

```
开发里程碑 2 开始时 ──→ 同步去办 ①营业执照（1~3天）
①执照下来 ──┬──→ ②买服务器（当天）──→ ③发起备案（等 1~3 周）
            └──→ ④注册正式账号（几天）
功能开发(里程碑2~3)做完时，备案大概率也好了 ──→ 进 Step 8~10 上线
```
